# Inicio Rápido

Pon en marcha tu mesa de ayuda inteligente en menos de 5 minutos.

> **Prerequisitos:** Solo necesitas un navegador web moderno y una cuenta de correo electrónico.

## 1. Crear tu cuenta

Visita [autoservice.pe/registro](/registro) y completa el formulario:

- **Nombre de empresa** — El nombre que verán tus usuarios en el portal de soporte
- **Email corporativo** — Será el login del administrador
- **Contraseña** — Mínimo 8 caracteres
- **RUC** (opcional) — Para facturación electrónica

## 2. Seleccionar un plan

Elige el plan que mejor se adapte a tu equipo:

| Plan | Agentes | Precio | Ideal para |
|------|---------|--------|------------|
| **Básico** | Hasta 3 | S/ 149/mes | Equipos pequeños |
| **Profesional** | Hasta 25 | S/ 349/mes | Equipos en crecimiento |
| **Enterprise** | Ilimitados | S/ 749/mes | Grandes organizaciones |

Todos los planes incluyen **14 días de prueba gratis** sin tarjeta de crédito.

## 3. Acceder a tu instancia

Una vez completado el registro, tu instancia estará lista automáticamente. Recibirás:

- **URL de acceso** — `tu-empresa.autoservice.pe`
- **Credenciales de administrador**
- **Guía de configuración** por email

## 4. Configuración inicial

Desde el panel de administración, configura lo esencial:

```
Panel Admin → Configuración → General
```

1. **Logo y marca** — Sube tu logo y personaliza colores
2. **Categorías** — Define las categorías de tickets (Redes, Software, Hardware, etc.)
3. **SLAs** — Configura tiempos de respuesta y resolución por prioridad
4. **Equipos** — Crea equipos y asigna categorías

## 5. Invitar a tu equipo

```
Panel Admin → Usuarios → Invitar Agente
```

Ingresa el email de cada agente. Recibirán una invitación para activar su cuenta.

**Roles disponibles:**
- **Administrador** — Acceso total a configuración y reportes
- **Agente** — Gestión de tickets y base de conocimiento
- **Agente Junior** — Solo gestión de tickets asignados

## 6. Crear tu primer ticket

¡Listo! Ya puedes crear tickets de tres formas:

- **Portal web** — Los usuarios finales envían tickets desde `tu-empresa.autoservice.pe/portal`
- **Email** — Configura un email de soporte que cree tickets automáticamente
- **API** — Integra con tus sistemas existentes vía REST API

La IA clasificará automáticamente cada ticket por prioridad, categoría y equipo asignado.
