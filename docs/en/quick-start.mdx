# Quick Start

Get your intelligent help desk up and running in less than 5 minutes.

> **Prerequisites:** You only need a modern web browser and an email account.

## 1. Create your account

Visit [autoservice.pe/registro](/en/register) and fill in the form:

- **Company name** — The name your users will see on the support portal
- **Corporate email** — This will be the admin login
- **Password** — Minimum 8 characters
- **RUC** (optional) — For electronic invoicing

## 2. Select a plan

Choose the plan that best fits your team:

| Plan | Agents | Price | Ideal for |
|------|--------|-------|-----------|
| **Basic** | Up to 3 | S/ 149/mo | Small teams |
| **Professional** | Up to 25 | S/ 349/mo | Growing teams |
| **Enterprise** | Unlimited | S/ 749/mo | Large organizations |

All plans include a **14-day free trial** with no credit card required.

## 3. Access your instance

Once registration is complete, your instance will be ready automatically. You'll receive:

- **Access URL** — `your-company.autoservice.pe`
- **Admin credentials**
- **Setup guide** via email

## 4. Initial configuration

From the admin panel, configure the essentials:

```
Admin Panel → Settings → General
```

1. **Logo & branding** — Upload your logo and customize colors
2. **Categories** — Define ticket categories (Networks, Software, Hardware, etc.)
3. **SLAs** — Configure response and resolution times by priority
4. **Teams** — Create teams and assign categories

## 5. Invite your team

```
Admin Panel → Users → Invite Agent
```

Enter each agent's email. They'll receive an invitation to activate their account.

**Available roles:**
- **Administrator** — Full access to settings and reports
- **Agent** — Ticket and knowledge base management
- **Junior Agent** — Only assigned tickets

## 6. Create your first ticket

You're ready! Create tickets in three ways:

- **Web portal** — End users submit tickets at `your-company.autoservice.pe/portal`
- **Email** — Configure a support email that creates tickets automatically
- **API** — Integrate with existing systems via REST API

AI will automatically classify each ticket by priority, category and assigned team.
