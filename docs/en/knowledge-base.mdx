# Knowledge Base

The Knowledge Base (KB) is AutoService's help article library. Users can find solutions before creating a ticket, and AI uses these articles to suggest responses.

## Creating articles

```
Admin Panel → Knowledge Base → New Article
```

Each article has:
- **Title** — Descriptive and clear
- **Content** — Rich editor with markdown, images, videos and code
- **Category** — Organized in logical sections
- **Tags** — For search and classification
- **Visibility** — Public (everyone) or Internal (agents only)
- **Status** — Draft or Published

### Best practices

1. **One problem, one article** — Keep articles focused
2. **Title as a question** — "How do I configure VPN?" is better than "VPN Configuration"
3. **Numbered steps** — Users scan, they don't read
4. **Include screenshots** — A picture is worth a thousand words
5. **Keep updated** — Review articles quarterly

## Intelligent search

KB search uses AI for:
- **Semantic search** — Finds articles by meaning, not just exact words
- **Synonyms** — "laptop" finds articles about "notebook" and "portable"
- **Auto-suggest** — As users type a ticket, suggests relevant articles

### Ticket deflection

When a user is creating a ticket, AutoService shows relevant articles automatically. If the article solves the problem, the user can close the form without creating the ticket.

> **Metric:** Average ticket deflection rate across our customers is 23%.

## KB Metrics

- **Most viewed articles** — Most consulted
- **Most helpful** — Best rated by users
- **Deflection rate** — Tickets avoided
- **Zero-view articles** — Candidates to remove or improve
- **No-result searches** — Topics needing new articles
